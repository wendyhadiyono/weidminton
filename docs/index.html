Berkenalan dengan WeidMinton

Mulai kelola klub badminton anda secara digital dengan kami. Seperti melacak kas, mengatur data anggota dan transaksi hingga melakukan abensi.

Tentang Unduh

Tentang

WeidMinton adalah aplikasi web yang membantu para bendahara atau admin untuk mengelola klub badminton mereka. Fitur utama kami adalah:

  1. Mengelola kas
  2. Mengelola anggota
  3. Mengelola transaksi anggota
  4. Mengelola transaksi bola
  5. Mengelola transaksi lapangan
  6. Melakukan absensi
